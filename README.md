# adambezance.com

Personal website of **Adam Bezance** — Microsoft MVP (M365 Copilot & Copilot Studio), AI Governance & Enablement.

Static site: plain HTML, CSS, and a little vanilla JavaScript. No build step — open `index.html` in a browser.

## Structure

- `index.html` — homepage
- `blog.html` — blog index (posts tagged Governance / Enablement)
- `blog/article-template.html` — copy this to create a new post
- `community.html`, `pluralsight.html`, `about.html`, `contact.html` — content pages
- `speaking.html` — built but currently unlinked, to be released later
- `css/styles.css` — the "Swiss Precision" design system
- `js/site.js` — shared nav / mobile menu / scroll reveal

Contact form is powered by Web3Forms.
